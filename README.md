# ef_shops3DText
This is a fork of the ef_shops GTAV Five M resource, modified to use J-textUI for prompt instead of the targeting system

MAKE SURE TO ENSURE IN YOUR SERVER.CFG LIKE THIS:

ensure j-3dtextui<br>
ensure ef_shops
